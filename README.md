# Custom Container (No Collections)

Учебный проект: реализация собственного контейнера, который может хранить произвольное количество объектов без использования встроенных коллекций (`ArrayList`, `LinkedList` и т.д.).

Контейнер поддерживает базовые операции: добавление, извлечение, изменение, удаление элементов.

## Требования 

- Хранить произвольное количество объектов
- Запрещено использовать встроенные коллекции Java
- Реализация через массив 
- Операции:
  - `add` - добавить элемент
  - `get` - получить элемент
  - `set` - изменить элемент
  - `remove` - удалить элемент

## API контейнера

- `add(value)` - добавляет элемент в конец
- `getElement(index)` - возвращает элемент по индексу
- `setElement(index, value)` - изменяет элемент по индексу
- `remove(index)` - удаляет элемент по индексу и сдвигает элементы
- `remove(value, isRemoveAll)` - удаляет элемент по значению и сдвигает элементы, если isRemoveAll = true, то удаляются все совпадения, иначе - только первое
- `getSize()` - текущее количество элементов
- `isEmpty()` - проверка на пустоту
- `clear()` - очищает все элементы
